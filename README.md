# Excel Import Demo

這是一個 ASP.NET MVC Excel 匯入與 SQL Server 資料處理範例，主要展示企業內部常見的資料匯入流程。

## 功能說明

- Excel 檔案上傳
- 資料欄位解析
- 欄位格式驗證
- 錯誤資料提示
- SQL Server 資料寫入
- 適合企業內部資料匯入、報表整合、ERP資料前處理

## 使用技術

- ASP.NET MVC
- C#
- .NET Framework
- SQL Server
- NPOI
- jQuery
- Bootstrap

## 適用情境

此範例適合用於：

- 訂單資料匯入
- 客戶資料匯入
- Excel 轉資料庫
- 企業內部後台系統
- 資料處理自動化流程

## 專案目的

展示如何透過 ASP.NET MVC 建立一個簡單但完整的 Excel 匯入流程，降低人工整理資料的時間與錯誤率。
